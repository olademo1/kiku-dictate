People team pilot for macOS

Fast voice-to-text without sending audio to a transcription API.

Dataiku Chirp runs Whisper large-v3 turbo locally on your Mac. It records a temporary audio file, transcribes it on-device, pastes the result where your cursor is, then deletes the temporary audio.

Built to be easy to explain.

Local transcription

Audio is processed by a bundled local Whisper runtime and model. The app does not call OpenAI, Apple Speech Recognition, or another transcription API.

No transcript history

Dataiku Chirp stores aggregate usage counts, not dictated text. Temporary audio is deleted after transcription or failure.

Open source pilot

The code is public for review. The pilot download is built from the same repository and bundled so teammates do not need Homebrew.

Install

Download the DMG, then drag to Applications.

Use the main download button on this page. Open the downloaded disk image, drag Dataiku Chirp into Applications, then open it from Applications. After that, macOS will ask for microphone access. Enable Accessibility only if you want automatic paste into the active app.

  1. Click Download Mac Installer.
  2. Open the downloaded DataikuChirp-macOS.dmg.
  3. Drag Dataiku Chirp into Applications.
  4. Open Dataiku Chirp from Applications.
  5. Choose your team during setup.
  6. Approve Microphone and, optionally, Accessibility.
